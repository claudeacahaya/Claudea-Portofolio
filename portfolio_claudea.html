04
Flask - Python - HTML - CSS - JSON

Dashboard Keuangan Organisasi

Dashboard menampilkan saldo akhir, total pemasukan, total pengeluaran, dan total kas anggota. Setiap pos dana memiliki ringkasan pemasukan, pengeluaran, dan saldo, sehingga bendahara dapat melihat kondisi keuangan tanpa menghitung manual.

  • Ringkasan saldo otomatis dari transaksi masuk dan keluar.
  • Pemisahan laporan berdasarkan pos dana.
  • Filter tanggal untuk mengecek transaksi tertentu.
  • Badge pemasukan dan pengeluaran agar tabel mudah dipindai.
  • Form tambah dan edit transaksi dengan field tipe, nominal, tanggal, keterangan, dan pos dana.
  • Penyimpanan data sederhana menggunakan file JSON.

Peran utama proyek ini adalah menjembatani kebutuhan administrasi organisasi dengan sistem digital yang sederhana, transparan, dan mudah dioperasikan.
